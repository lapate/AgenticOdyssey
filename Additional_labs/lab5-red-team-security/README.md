# Lab 5 – Red Team Security Testing

## Scenario

In this lab, you will perform **red team security scans** on the RotisseriePlannerAgent to proactively identify vulnerabilities by simulating adversarial attacks.

## Prerequisites

- Completed **Lab 1** through **Lab 4**
- Python 3.10+
- Azure CLI authenticated: `az login`
- `AZURE_OPENAI_API_KEY` set in `.env`
- AI Foundry project in a supported region (East US, Sweden Central, East US 2)

## Instructions

Run the notebook `Lab5_Red_Team_Security.ipynb` in this folder.
